# ผู้พิทักษ์เวทมนตร์แห่งสิทธิ: กติกา Alpha v3

สถานะ: digital playtest draft  
สำรับหลัก: v3 คำเดี่ยว 53 ใบ  
ผู้เล่นแนะนำ: 2 ทีม, ปรับเป็น 3 ทีมได้  
เวลาเล่นเป้าหมาย: ประมาณ 5-10 นาทีต่อรอบ

## เป้าหมายของเกม

แต่ละทีมพยายามอ่านให้ออกว่า Venn 2 วงบนโต๊ะแทนสนธิสัญญาใด และลงการ์ดคำให้ถูกตำแหน่ง ทีมที่ทำให้การ์ดหมดมือก่อนชนะรอบนั้น

เกมเล่นทั้งหมด 3 รอบ แล้วนับแต้มรวม

## อุปกรณ์

- การ์ดคำ v3 จำนวน 53 ใบ
- พื้นที่ Venn 4 ช่อง
  - วงซ้าย
  - ตรงกลาง
  - วงขวา
  - นอกวงทั้งหมด
- token ทายวงของแต่ละทีม
  - token ฝั่งซ้าย 1 อัน
  - token ฝั่งขวา 1 อัน
- ตัวนับหลุดการป้องกัน 0/8

## ตั้งค่าเริ่มต้น

- จำนวนทีม: 2 ทีม
- การ์ดเริ่มต้น: ทีมละ 7 ใบ
- หลุดการป้องกัน: เกิน 8 ใบ ทุกทีมแพ้รอบนั้น
- รอบแรกใช้คู่สอน: CRC + ICESCR
- ก่อนเริ่มรอบ ผู้นำเกมเปิดตัวอย่างวงซ้าย / ตรงกลาง / วงขวา อย่างละ 1 ใบถ้ามี แล้ววางให้ถูกตำแหน่งเป็นข้อมูลตั้งต้น
- ระบบสุ่มเฉพาะคู่ Venn ที่มีการ์ดซ้าย ขวา และตรงกลางพอให้เห็นหลักฐานระหว่างเล่น

## วิธีเล่นในแต่ละตา

1. ทีมปัจจุบันเลือกการ์ด 1 ใบจากมือ
2. ทีมแตะตำแหน่งที่คิดว่าถูก
   - วงซ้าย
   - ตรงกลาง
   - วงขวา
   - นอกวงทั้งหมด
3. ระบบเฉลยทันที
   - ถ้าถูก การ์ดอยู่ตำแหน่งนั้น
   - ถ้าผิด ระบบย้ายการ์ดไปตำแหน่งที่ถูก และทีมนั้นจั่วเพิ่ม 1 ใบ
4. ถ้าทีมวางถูก ทีมเดิมได้ลงต่อ แต่ลงต่อเนื่องได้ไม่เกิน 2 ใบ
5. ถ้าทีมวางผิด ส่งตาให้ทีมถัดไป

## นอกวงและหลุดการป้องกัน

ถ้าทีมทายว่า `นอกวงทั้งหมด` แล้วถูก จะไม่เกิดโทษ

ถ้าทีมวางผิด และคำตอบที่ถูกคือ `นอกวงทั้งหมด` การ์ดนั้นจะนับเป็น `หลุดการป้องกัน`

เมื่อหลุดการป้องกันเกิน 8 ใบ ทุกทีมแพ้รอบนั้นทันที รอบนั้นไม่มีทีมได้แต้มชนะรอบ แต่ยังเฉลย token และนับแต้ม token ตามปกติ

## Token ทายวง

แต่ละทีมมี token 2 อัน:

- 1 อันสำหรับทายวงซ้าย
- 1 อันสำหรับทายวงขวา

ทีมเลือกสนธิสัญญาจากคลัง แล้ววาง token ลงฝั่งซ้ายหรือขวาได้ในตาของตัวเอง

ทีมสามารถย้าย token ของตัวเองได้ก่อนจบรอบ

คลัง token แสดงเป็นระบบเดียวกันทุกสนธิสัญญา:

- ตัวย่อสนธิสัญญา เช่น CRC, ICERD
- ชื่อสิทธิย่อ เช่น เด็ก, ไม่เหยียดที่มา
- คำเปรียบเทียบสั้น ๆ เพื่อช่วยให้มือใหม่แยกภาพรวมของแต่ละสิทธิ

## การคิดแต้ม token

เฉลย token ตอนจบรอบ

- ทายถูกและเป็นทีมแรกของฝั่งนั้น: +1
- ทายถูกแต่ไม่ใช่ทีมแรก: 0
- ไม่วาง token: 0
- ทายผิด: -1

## คำใบ้

เมื่อหลุดการป้องกันถึง 2 ใบ ผู้เล่นเลือกอ่านคำใบ้ได้ 1 วง

เมื่อหลุดการป้องกันถึง 4 ใบ ผู้เล่นเลือกอ่านคำใบ้อีกวงได้

ไม่มีคำใบ้ก่อนเริ่มเกม

แต่ละสนธิสัญญามีคำใบ้ 3 ชุด ระบบจะสุ่มใช้ 1 ชุดต่อวงในแต่ละรอบ เพื่อลดการจำคำตอบจากคำใบ้เดิมซ้ำ ๆ

## การจบรอบ

รอบจบเมื่อเกิดอย่างใดอย่างหนึ่ง:

- ทีมใดทีมหนึ่งการ์ดหมดมือ: ทีมนั้นชนะรอบและได้ +1 แต้ม
- กองจั่วหมด: ทุกทีมแพ้รอบนั้น
- หลุดการป้องกันเกิน 8 ใบ: ทุกทีมแพ้รอบนั้น

หลังจบรอบ ให้เฉลยว่าวงซ้ายและวงขวาคือสนธิสัญญาใด แล้วนับแต้ม token

## การชนะแมตช์

เล่นครบ 3 รอบ

แต้มรวมมาจาก:

- ชนะรอบ: +1
- token: +1 / 0 / -1 ตามผลทายวง

ทีมที่มีแต้มรวมสูงสุดชนะ ถ้าแต้มเท่ากันถือว่าเสมอ

## ความสามารถกลาง

### เปลี่ยนการ์ด

เลือกการ์ด 1 ใบจากมือ ทิ้งการ์ดนั้น แล้วจั่วใหม่ 1 ใบ ใช้ได้ 1 ครั้งต่อหนึ่งตา

### กู้คืน

ใช้ได้ 1 ครั้งต่อรอบ เมื่อทุกทีมเห็นด้วย

ทุกทีมจั่วเพิ่ม 1 ใบ แล้วเอาการ์ดหลุดการป้องกันออก 1 ใบ

## จุดที่ facilitator ควรถาม

หลังมีการวางการ์ดครบประมาณ 10 ใบ ถ้ามีทีมวางผิด ให้ถามสั้น ๆ:

- ตอนแรกทีมคิดว่าอะไรทำให้การ์ดนี้อยู่ช่องนั้น
- หลังเห็นเฉลยแล้ว หลักคิดควรเปลี่ยนอย่างไร
- การ์ดนี้ช่วยบอกอะไรเกี่ยวกับขอบเขตของสนธิสัญญา

## สิ่งที่ควรสังเกตระหว่าง playtest

- ผู้เล่นเข้าใจการวางซ้าย / กลาง / ขวา / นอกวงหรือไม่
- token ทำให้ผู้เล่นกล้าเดาวงเร็วขึ้นหรือไม่
- คำเดี่ยวใน v3 สื่อชัดพอหรือยัง
- หลุดการป้องกัน 8 ตึงเกินไปหรือหลวมเกินไป
- รอบหนึ่งจบใน 10 นาทีหรือไม่
- มีการ์ดใบใดที่ผู้เล่นเถียงเฉลยมากเป็นพิเศษ
